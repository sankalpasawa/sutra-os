### Parsing caveat (known limitation)

This scanner scrapes markdown tables in PROTOCOLS.md and MANIFEST-v*.md by
pipe columns + regex. Changes to table shape (column count, indentation,
rowspans) may produce false DRIFT or false OK without any real semantic
change. Preferred remedy is a structured source (yaml/jsonl registry) — see
follow-up backlog.
